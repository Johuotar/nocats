Cave Flying Game

Use arrow keys to control a firepowered flying triangle. Avoid the asteroids (that somehow ignore gravity) and find the container on the level. You can use spacebar to activate a tractorbeam that you can with some difficulty use to get the container to the floating base, and win the game.
